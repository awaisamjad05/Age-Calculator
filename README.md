# AG-Tech Chronos — Premium Age & Cosmic Portal

Chronos is a handcrafted, state-of-the-art, single-page age dashboard built using vanilla HTML5, CSS3, and JavaScript. It features no external AI dependencies, relying on precise astronomical and calendar mathematics to deliver an immersive, premium user experience.

## ✨ Premium Features

- **Real-Time Chronological Ticking**: Watch your age increment to the exact second in real time.
- **Multitasking Profiles**: Create, edit, and delete multiple age profiles (family, friends, pets) stored locally inside your browser (`localStorage`).
- **Accent Customization**: Choose between 5 vibrant ambient themes (Indigo, Rose, Emerald, Amber, Cyan) with active glow reflections.
- **Progress Circle Countdown**: A gorgeous circular countdown indicating exactly how many days, hours, minutes, and seconds remain until your next birthday.
- **Life Journey Progress**: A fluid progress bar mapping out your lifecycle (based on an 80-year lifespan) with active/completed milestones.
- **Zodiac & Astro Signposts**: Automatically calculates both Western Zodiac (with elements and date ranges) and Chinese Zodiac (with cycle animals and elements).
- **Biological Milestones**: Real-time estimates of total heartbeats taken, breaths inhaled, and hours slept, alongside the cultural folklore associated with the day of the week you were born.
- **Cosmic Planet Explorer**: Discover how old you are in Martian, Venusian, or Jovian years, and see countdowns for your next planetary birthdays.
- **Time Travel Simulator**:
  - *Age Simulator*: Find out exactly how old you'll be on any calendar date in the future.
  - *Milestone Calculator*: Input milestones (e.g., 10,000 days lived or 1 billion seconds lived) and find the exact day they will be reached.
- **Celebration Effects**: Built-in canvas-confetti bursts on profile creation and birthday countdown milestones.

## 🚀 How to Run Locally

1. Double-click the `index.html` file to open it directly in any modern web browser.
2. Alternatively, serve the directory using any local server:
   ```bash
   # Using python
   python -m http.server 8000
   
   # Using node (npx)
   npx browser-sync start --server --files "*.html, *.css, *.js"
   ```
3. Enjoy exploring your cosmic timeline!
